## 空闲进程设置

init_idle()函数的作用是将当前正在执行的进程（current为一个指向当前正在执行的进程的任务结构体）正式标记为当前CPU的空闲进程，并将该进程与当前
CPU 的 rq（运行队列）中的 idle 指针关联，确保当该 CPU
没有其他进程可运行时，会自动切换回到这个任务来执行“空闲循环”。空闲进程的调度类被设定为
idle_sched_class，这是系统中优先级最低的调度类。只有当没有其他任务可执行时，才会轮到它运行。

函数idle_thread_set_boot_cpu()主要用于在系统启动或 CPU
热插拔过程中，将当前的 引导 CPU
标记为拥有一个有效的空闲线程。它把per-cpu变量中的名为 idle_threads 的
任务结构体赋值为当前正在执行的任务（current全局变量），即将当前正在执行初始化代码的进程（PID
0，即 idle 进程）设置为该 CPU 的 空闲 任务。
