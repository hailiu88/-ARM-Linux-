## 文件查看命令

**cat**------ 显示文件内容，格式为：

    cat [选项] 文件

常用选项包括：

-n 给所有行编号（包括空行）

-b 给非空行编号

-s 压缩连续空行

-E 显示行尾 \$

-T 将 tab 显示为 \^I

-A 显示所有不可见字符（等于 -vET）

-v 显示不可打印字符

示例：

    $ cat file，将文件file内容显示到屏幕

    $ cat -n file，将文件file内容显示到屏幕，对内容逐行（包括空行）编号

**more**------逐页显示文件内容，格式为：

    more [选项] 文件

常用选项包括：

-d 使用提示信息而不是响铃，例如 [Press space to continue, \'q\' to quit.]

-c 清屏后显示新一页，减少屏幕滚动

-f 统计文件行数时把长行折算为单行（对长行处理）

-l 忽略表单换页字符（\^L）

-s 压缩连续空行

-u 去掉下划线的显示（去掉 \_）

+n 从第 n 行开始显示文件

在运行过程中，可以采用下列操作：

空格 下一页

Enter 下一行

b 上一页

/pattern 向下搜索字符串

n 重复上次搜索

q 退出

= 显示当前行号

f 前进一屏（部分系统）

Ctrl+D 前进半屏

Ctrl+B 后退一屏

less（推荐）

示例：

    $ more file1

**less**------ 高级分页查看命令，格式为：

    less [选项] [文件\...]

常用选项包括：

-N 显示行号

-S 不折行（长行水平滚动）

-X 不清屏（退出 less 后保留内容）

-i 搜索忽略大小写

-I 搜索强制大小写敏感（除非全大写）

-r 显示原始控制字符（如颜色）

-R 显示 ANSI 彩色序列（推荐）

+F 类似 tail -f 实时查看文件

+G 打开时跳到文件末尾

在运行期间可执行如下操作：

空格 向下翻一页

Enter 向下翻一行

b 向上翻一页

y 向上翻一行

G 跳转到文件末尾

1G 或 g 跳转到文件开头

/pattern 向下搜索

?pattern 向上搜索

n 重复上次搜索

N 反向重复搜索

q 退出

示例：

    $ less -N file1

**head** ------查看文件开头若干行命令，格式为：

    head [选项] 文件...

常用选项包括：

-n NUM 显示前 NUM 行

-c NUM 显示前 NUM 字节

-q 多文件时不显示文件名

-v

示例：

    $ head -n 10 file

**tail**------查看文件尾若干行命令，格式为：

    tail [选项] 文件...

常用选项包括：

-n NUM 显示后 NUM 行

-c NUM 显示后 NUM 字节

-f 实时查看文件追加（类似日志监控）

-F 类似 -f，支持文件轮转（更安全）

-q 多文件时不显示文件名

-v 总是显示文件名（多文件）

示例：

    $ tail -n 20 file

    $ tail -f logfile
