## 磁盘与系统信息

**df**（disk free）**------**查看文件系统磁盘空间使用情况命令，格式为：

    df [选项] [文件或目录]

常用选项包括：

选项 说明

-a 列出所有文件，包括伪文件、重复文件及不可访问文件

-h 自动单位（推荐）

-H 1000 进制单位

-k KB 显示

-m MB 显示

-B SIZE 指定单位

-i 显示inode使用情况

-T 显示文件系统类型

-X 排除要显示的文件类型

-l 只显示本地文件

示例：

    $ df -m

    $ df -B 1G

    $ df -h /var/log，查看目录/var/log所在分区

    $ df -t ext4，只查看ext4文件系统

**du**（disk usage）------统计文件或目录实际占用的磁盘空间命令，格式为：

    du [选项] [文件或目录]

常用选项详解包括：

选项 说明

-h 自动单位（推荐）

-H 1000 进制单位

-k KB

-m MB

-B SIZE 指定块大小

-s 只显示总计

\--max-depth=N 显示指定层级

\--exclude 排除指定文件或目录

\--apparent-size 统计文件逻辑大小，du默认统计实际占用磁盘块

-a： all（包括文件）

示例：

    $ du -m /home

    $ du -B 1G

    $ du -h --max-depth=1

    $ du -h --max-depth=2 /var

    $ du -h --exclude=\*.log，不包括后缀为.log的文件

    $ du -h --exclude=/home/test，排除目录/home/test

    $ du -h --apparent-size，统计文件逻辑大小（而非占用块数）

    $ du -ah，统计所有文件（包含子目录文件）

    $ du -sh * |sort -hr|head -20，排序查看最大目录

free------查看 系统内存和交换分区使用情况命令，格式为：

    free [选项]

常用选项包括：

选项 说明

-h 自动单位

-b 字节

-k KB

-m MB

-g GB

-s N 实时刷新

-t 显示总内存与交换区

示例：

    $ free -h，以可读方式显示所有内存与缓冲区使用情况

    $ free -h -s 2，每2秒刷新一次

**uname**（unix name）------显示系统内核与操作系统信息命令，格式为：

    uname [选项]

常用选项包括：

选项 说明

-a 显示全部信息

-s 内核名称

-n 主机名

-r 内核版本

-v 内核编译版本

-m 硬件架构

-p 处理器类型

-i 硬件平台

-o 操作系统

示例：

    $ uname -r，显示内核版本号

    $ uname -v，显示编译信息
