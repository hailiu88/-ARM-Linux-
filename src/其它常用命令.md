## 其它常用命令

**echo**------显示字符串，格式为：

    echo [选项] 字符串

常用选项包括：

```text
选项        说明

-n          不换行

-e          解析转义字符

-E          不解析转义字符（默认）
```

转义字符有：

```text
转义符      含义

`\n`        换行

`\`t        制表符

`\\`        反斜杠

`\r`        回车

`\b`        退格

`\a`        响铃

`\c`        终止输出
```

示例：

    $ echo david，在屏幕上打印david字符串

    $ name="My name is David"

    $ echo $name，在屏幕上显示变量name的值，即显示My name is David

    $ `echo "Hello \nWorld"`，在屏幕上分两行分别显示Hello和World

当转义字符包含在双引号之间时，要发生转义。

`''` 与 `""` 的区别:

在 Linux Shell（如 bash）中，单引号 `''` 和双引号 `""`
都用于引用字符串，但行为不同。

核心区别：

```text
特性        `'单引号'`      `"双引号"`

变量展开    ❌ 不展开       ✅ 展开

命令替换    ❌ 不执行       ✅ 执行

转义符处理  ❌ 不解析       部分解析

通配符展开  ❌ 不展开       ❌ 不展开
```

**ln**------创建链接文件命令，格式为：

`ln [选项]` 源文件 目标文件

选项包括：

```text
选项    说明

-s      创建符号链接

-f      强制覆盖

-i      覆盖前提示

-v      显示创建过程

-n      目标为符号链接时不解引用

-d      允许对目录创建硬链接（超级用户）
```

链接分硬链接和符号链接。硬链接指向同一个inode，删除其中一个不影响其余目标或源文件。符号链接可视为快捷链接，删除源文件后，目标文件变为悬空链接。

示例：

    $ ln file.txt file_link.txt，为file.txt建立链接文件file_link.txt。默认为硬链接

    $ ln -s file.txt soft_link.txt，建立符号链接或软链接

**export**------将变量设置为环境变量命令，格式为：

    export [选项] [变量名]

常用选项为：

```text
选项    作用

-p      显示已导出变量

-n      取消导出

-f      导出函数
```

示例：

    $ export -P

    $ export PATH=\$PATH:/opt/bin

    $ export JAVA_HOME=/usr/lib/jvm/java-11

**env**------显示或临时修改环境变量命令，格式为：

    env [选项] [变量=值 ...] [命令 [参数...]]

常用选项包括：

```text
选项            作用

无参数          显示所有环境变量

VAR=val         设置临时变量

-i              清空环境

-u VAR          删除变量

`--help`        帮助

`--version`     版本
```

示例：

    $ env，显示环境变量

    $ env DEBUG=1 ./program，设置变量DEBUG值为1，并执行当前目录下的程序program

    $ env -i bash，清空环境变量并运行bash

    $ env -u PATH bash，清除PATH变量后启动bash
