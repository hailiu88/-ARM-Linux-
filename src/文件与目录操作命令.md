## 文件与目录操作命令

**ls** ------ 文件列表命令，格式为：

    ls [选项] 目录或文件名

常用选项包括：

 -l 长格式显示

 -a 显示隐藏文件

 -h 人类可读大小

 -t 按时间排序

 -r 反向排序

-R 递归显示

 -i 显示 inode

 -S 按文件大小排序

 -d 显示目录本身

示例：

    $ ls

    $ ls -l

    $ ls -lh

其中，-l 表示列出详细信息，-h表示以
人类可读方式显示大小，必须与l配合使用，-a
表示显示隐藏文件，-t表示按时间排序，-r表示 反向排序。

**pwd** ------ 显示当前目录命令，格式为：

    pwd

**cd** ------ 切换目录命令，格式为：

    cd [选项] 目录名

常用选项包括：

 -L 逻辑路径（默认）

 -P 物理路径（解析真实路径）

 -e 与 -P 配合，失败时报错

示例：

    $ cd /home，跳到根目录下的home目录

    $ cd ..，返回上一级目录

    % cd \~，切换到当前用户的主目录。

    $ cd --，切换到上一次所在目录

**mkdir** ------ 创建目录，格式为：

    mkdir [选项] 目录名

常用选项包括：

 -p 递归创建父目录

 -v 显示创建过程

 -m 指定权限

示例：

    $ mkdir dir1，创建目录dir1

    $ mkdir -p a/b/c，递归创建目录，
其中-p为父目录选项，如果父目录a不存在，则创建父目录a，如果父目录b不存在，则创建目录b。

**rmdir** ------ 删除空目录，格式为：

    rmdir [选项] 目录名

**rm** ------ 删除文件或目录，格式为：

    rm [选项] 目录或文件名

常用选项包括：

-r / -R 递归删除目录

-f 强制删除（不提示）

-i 删除前提示

-v 显示删除过程

-d 删除空目录

-I 删除多文件时询问一次

示例：

    $ rm file， 删除文件

    $ rm -r dir，删除目录（递归删除文件）

    $ rm -rf dir， 强制删除目录

-rf 慎用。

**cp** ------ 复制文件或目录。格式为：

    cp [选项] 源文件或目录 目标文件或目录

选项包括选项：

-r / -R 递归复制目录

-a 归档模式

-f 强制覆盖

-i 覆盖前提示

-n 不覆盖已存在文件

-v 显示复制过程

-u 仅复制更新文件

-p 保留权限、时间、所有者

\-\-preserve=ATTR 指定保留属性

\-\-no-preserve=ATTR 不保留某些属性

示例：

    $ cp file1 file2，复制file1到file2

    $ cp -r dir1 dir2，复制目录dir1到dir2，包含子目录

**mv** ------ 移动/重命名，格式为：

    mv [选项] 原文件或目录 新文件或目录名

选项包括选项：

-i 覆盖前提示

-f 强制覆盖

-n 不覆盖已有文件

-v 显示过程

-u 只移动更新文件

-t DIR 指定目标目录

-T 将目标视为普通文件

示例：

    $ mv file /tmp
