## 停用中断

在完成系统初始化之前，不能够让中断发生。停用中断的函数为local_irq_disable()，虽然这个函数具有c语言格式，但由于停用中断指令依赖于硬件本身，因此，实际的停用中断函数是用汇编语言编写的。通过宏定义语句，local_irq_disable()函数调用raw_local_irq_disable()，而宏定义又把raw_local_irq_disable()定义为arch_local_irq_disable()。每一类CPU都会提供自己的arc_local_irq_disable()函数。由于arch_local_irq_disable()是以在c语言中嵌入汇编语言的格式编写，其调用格式符合c语言函数调用格式，因此可以在c语言程序中调用停用中断函数。

对arm处理器而言，arch_local_disable_irq()定义在linux/arch/arm/include/asm/irqflags.h文件中，其定义为：

```
static inline void arch_local_irq_disable(void)

{

asm volatile(" cpsid i @ arch_local_irq_disable"

:

:

: "memory", "cc");

}
```
