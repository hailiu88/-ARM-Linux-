## 压缩与解压

**tar**------打包解包命令，格式为：

    tar [选项] 文件名 [文件或目录]

常见结构为：

    tar -选项 归档文件 要打包的文件

常用选项包括：

-c 创建归档（create）

-x 解包（extract）

-t 查看归档内容（table）

-r 向归档追加文件

-u 仅追加较新文件

-f 指定归档文件名（必须），-f 后面必须紧跟归档文件名

压缩相关选项：

压缩方式 文件后缀

-z gzip .tar.gz

-j bzip2 .tar.bz2

-J xz .tar.xz

-a 自动识别压缩类型 自动

示例：

    $ tar -cvf archive.tar dir/，创建 tar 包（不压缩）

其中，c表示创建，v 表示显示过程，f 用以指定文件

    $ tar -czvf archive.tar.gz dir/，创建 gzip 压缩包

    $ tar -cJvf archive.tar.xz dir/，创建 xz 压缩包（高压缩率）

    $ tar -xvf archive.tar，解压 tar 包

    $ tar -xzvf archive.tar.gz，解压 tar.gz

    $ tar -xzvf archive.tar.gz -C /target/path，解压到指定目录

    $ tar -tvf archive.tar，查看归档内容

常用控制选项：

选项 意义

-v 显示详细过程

-C 切换目录

\-\-exclude 排除文件

-p 保留权限

\--strip-components=n` 去除前 n 层目录

    $ $tar -czvf backup.tar.gz /project --exclude="*.log"`，排除文件示例

    $ tar -xvf archive.tar file.txt，只解压某个文件

    $ tar -czvf backup.tar.gz .，打包当前目录

解包示例：

    $ tar -xvf file.tar

    $ tar -czvf file.tar.gz dir

    $ tar -xzvf file.tar.gz

    $ zip -r file.zip dir

    $ unzip file.zip
