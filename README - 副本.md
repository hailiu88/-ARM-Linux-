## 🗺️ 本书核心知识地图

本书将带你走过 Linux 启动的五大核心阶段：

1.  **⚡ 硬件上电与 RomBOOT**：按下电源键后，ARM 处理器如何执行第一条指令，BootROM 与复位向量的底层机制。
2.  **引导加载程序（SPL / U-Boot）**：从低级内存初始化到加载 Linux 内核镜像（Image/zImage）与设备树（DTB）的完整过渡。
3.  **解压与内核解耦（Kernel Startup）**：内核自解压阶段，跳转到未压缩内核入口，开启 MMU 与页表建立的过程。
4.  **内核核心初始化（C 语言世界）**：`start_kernel()` 深度解析。包括内存管理（Page Allocator Pipeline）、中断向量表设置、调度器初始化及驱动加载。
5.  **黎明破晓（User Space）**：内核态到用户态的惊艳跳转，`rest_init()` 派生出 1 号进程，全面解析 `init` 进程（systemd/busybox init）如何接管世界。
